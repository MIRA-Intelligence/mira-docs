# {{PROJECT_UI_NAME}} 功能导航

`{{PROJECT_UI_NAME}}` 是 `{{PROJECT_CORE_NAME}}` 的可视化前端，覆盖 **创建项目 → 跑实验 → 看证据 → 导出交付物** 的完整闭环。它本身不存数据，所有真相都落在 `~/.mira/workspace/`，UI 只做展示与触发。

```mermaid
flowchart LR
    Home[首页/项目列表] --> Project[项目主页]
    Project --> Phase{当前阶段}
    Phase -- Research --> R[Research 面板<br/>背景/文献/问题定义]
    Phase -- Experiment --> E[Experiment 列表<br/>状态/指标/工件]
    Phase -- Result --> Res[Result 导出中心<br/>报告/论文/PPT]
    R --> Settings[运行参数<br/>run_mode / profile / contract]
    E --> Settings
    Res --> Settings
    Settings --> Sync[实时同步<br/>WS+REST]
```

## 一些约定

- **快捷键**：`⌘/Ctrl + K` 打开命令面板（搜项目/实验/动作）；`⌘/Ctrl + Enter` 在编辑框里发送。
- **状态色**：`pending` 灰、`running` 蓝（带动画）、`completed` 绿、`failed` 红、`修复中` 黄。
- **"刷新计划" 按钮**：UI 显示与 `task_plan.json` 不一致时，点它强制重拉一次完整计划。
